📖 INSTRUCCIONES RÁPIDAS PARA WINDOWS
=====================================

1. EXTRAER ARCHIVOS
   - Descomprime la carpeta Academia_dev en:
   C:\Users\[TuUsuario]\OneDrive\Documentos\Academia_dev
   
   La estructura debe ser:
   C:\Users\josue\OneDrive\Documentos\Academia_dev\
   ├── index.html
   ├── README.md
   ├── css\
   │   ├── styles.css
   │   └── pdf-viewer.css
   └── js\
       ├── app.js
       ├── pdf-handler.js
       └── all-scripts.js


2. ABRIR EN NAVEGADOR
   
   OPCIÓN A: Doble clic (más rápido)
   - Haz doble clic en index.html
   - Se abrirá automáticamente en tu navegador
   
   OPCIÓN B: Servidor local (recomendado)
   - Abre Command Prompt (Cmd) o PowerShell
   - Navega a la carpeta:
     cd C:\Users\josue\OneDrive\Documentos\Academia_dev
   
   - Si tienes Python 3:
     python -m http.server 8000
     Luego abre: http://localhost:8000
   
   - Si tienes Node.js:
     npx http-server
     Luego abre: http://localhost:8080


3. USAR LA APLICACIÓN
   
   📝 CREAR NOTAS CON PDF:
   - Click en "Notas" en el menú lateral
   - Click en "➕ Nueva Nota"
   - Completa título y contenido
   - En "Adjuntar PDF", haz click o arrastra un PDF
   - El PDF aparecerá como adjunto
   - Haz click en el adjunto para verlo
   
   🔍 ZOOM EN PDF:
   - Una vez abierto el PDF, usa:
     🔍+ para ampliar
     🔍− para alejar
   - La escala se muestra en porcentaje


4. CONFIGURACIÓN
   
   TEMA:
   - Haz click en ☀️ para cambiar tema (claro/oscuro)
   
   SONIDOS:
   - Haz click en 🔔 para activar/desactivar sonidos
   
   PERFIL:
   - Ve a Configuración > Mi Perfil para personalizar


5. GUARDAR DATOS
   
   Los datos se guardan automáticamente en:
   - LocalStorage del navegador
   
   IMPORTANTE:
   - Los datos solo se guardan en TU navegador
   - Si limpias caché, se perderán
   - Haz backup descargando tus notas regularmente


6. PRIMEROS PASOS RECOMENDADOS
   
   1. Crea tu perfil (Mi Perfil)
   2. Agrega un nuevo semestre (Semestres)
   3. Agrega tus materias (Materias)
   4. Agrega tu horario (Mi Horario)
   5. Empieza a crear notas con PDFs


7. SOLUCIÓN DE PROBLEMAS
   
   "No se abre el PDF"
   → Verifica que sea un PDF válido
   → Intenta con otro navegador (Chrome, Firefox, Edge)
   
   "No se guardan los datos"
   → No uses navegación privada/incógnito
   → Verifica que LocalStorage esté habilitado
   
   "El navegador dice que no puedo abrir archivos"
   → Usa la OPCIÓN B (servidor local) en lugar de doble clic


8. ARCHIVOS IMPORTANTES
   
   index.html
   - El archivo principal que debes abrir
   
   css/styles.css
   - Estilos principales (tema, colores, diseño)
   
   css/pdf-viewer.css
   - Estilos específicos del visualizador de PDF
   
   js/app.js
   - Lógica de la aplicación
   
   js/pdf-handler.js
   - Manejo de PDFs (lo más importante para ti)
   
   js/all-scripts.js
   - Scripts adicionales de la aplicación


9. PERSONALIZACIÓN
   
   Cambiar color principal:
   - Abre css/styles.css
   - Busca --accent: #7c6aff;
   - Cambio el valor (ej: #ff6b6b para rojo)
   - Guarda y recarga la página
   
   Cambiar nombre de la app:
   - En index.html, busca <title>Academia</title>
   - Cambia "Academia" por lo que quieras
