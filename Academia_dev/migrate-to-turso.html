🔄 Migración de Supabase a Turso

¿Qué hace esta herramienta?

Esta herramienta migra tus datos académicos desde Supabase a Turso:

  • Lee datos de la tabla user_data en Supabase
  • Crea la tabla user_data en Turso si no existe
  • Inserta los datos en Turso separados por user_id
  • Configura tu app para usar Turso automáticamente

Configuración de Turso

Si aún no tienes una cuenta en Turso:

  1. Ve a turso.tech y crea una cuenta gratis
  2. Crea una nueva base de datos (ej: "academia")
  3. Copia la URL de la base de datos
  4. Genera un token de autenticación

⚠️ Notas importantes

  • La migración es unidireccional: Supabase → Turso
  • Después de migrar, puedes mantener Supabase solo para auth
  • Los datos en Supabase no se eliminan automáticamente
  • Si algo falla, puedes desactivar Turso borrando las credenciales del localStorage
