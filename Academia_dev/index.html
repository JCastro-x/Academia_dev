FUNCIONALIDADES

Todo lo que necesitás
para el semestre.

☁️
Sync multi-dispositivo en tiempo real
Abrí la app en tu PC y en el celular al mismo tiempo. Los cambios se propagan automáticamente entre dispositivos vía Supabase. Un sistema de guard evita conflictos si modificás desde dos lugares a la vez.
Supabase Row Level Security Debounced sync Conflict guard
⏱️
Pomodoro anti-throttle
Timer en Web Worker con timestamps absolutos. No importa si cambiás de pestaña o el browser lo congela — el tiempo siempre es preciso.
Web Worker Picture-in-Picture
🎯
Calificaciones por zona
Ingresá notas por parcial, laboratorio y final. El promedio ponderado por créditos se calcula solo, con proyección de qué necesitás para aprobar.
📝
Notas con OCR
Adjuntá imágenes y PDFs a tus notas. Tesseract.js extrae el texto de documentos escaneados automáticamente.
Tesseract.js PDF.js
🃏
Flashcards
Tarjetas de estudio por materia con flip animado. Modo repaso con progreso por sesión para memorizar eficientemente.
📱
PWA instalable
Instalala en Android, iOS o escritorio como app nativa. Service Worker con cache inteligente — funciona offline.
Network First Cache First
