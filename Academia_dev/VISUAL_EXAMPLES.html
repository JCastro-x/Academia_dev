1️⃣ SIDEBAR DRAWER (Mobile)

Comportamiento en Mobile (< 768px)
El sidebar se convierte en drawer que se desliza desde la izquierda. Requiere overlay clickeable para cerrar.
Sidebar siempre visible
Ocupa 240px
Main content 60% ancho
Sidebar escondido (left: -100%)
Hamburger para abrir
Main content 100% ancho
Overlay semi-transparente
/* CSS */ .sidebar { transform: translateX(-100%); transition: transform .25s; } .sidebar.open { transform: translateX(0); } /* HTML */ <div id="sidebar-overlay"></div> <div id="sidebar" class="sidebar">...</div>
/* JavaScript */ function openSidebarDrawer() { sidebar.classList.add('open'); overlay.classList.add('visible'); } function closeSidebarDrawer() { sidebar.classList.remove('open'); overlay.classList.remove('visible'); }
Interacciones Esperadas
El usuario debe poder:
✅ Click hamburger → abre sidebar
✅ Click overlay → cierra sidebar
✅ Click nav item → navega + cierra
✅ Press ESC → cierra sidebar
✅ Resize > 768px → cierra automático
Importante: El drawer NO debe tener scroll horizontal. Usar overflow-y: auto si contenido es muy largo.
