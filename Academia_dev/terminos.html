2. Descripción del Servicio

Academia Dev es un dashboard académico diseñado para ayudarte a organizar tu semestre universitario. La Plataforma permite:

  • Gestionar calificaciones y materias
  • Organizar tareas y eventos académicos
  • Crear notas y apuntes
  • Usar un temporizador Pomodoro
  • Sincronizar datos entre dispositivos
